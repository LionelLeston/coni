This is an R project containing the data, analyses, and results for contract #3000704376 ("Analysis of bird survey data to refine monitoring designs and survey protocols") between the Canadian government (represented by Kevin Hannah) and myself (Lionel Leston), who was hired to run these analyses for this contract.

The scripts, analyses, and results will be stored in this R project for transparency and reproducibility.

The data for analysis is stored in the Access database "CONIlat_Mstr.mdb" and consists of 6 tables:

"BoomEvents"
"Moon"
"PeentEvents"
"SitesVisits"
"Temperatures"
"Times"
